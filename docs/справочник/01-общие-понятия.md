# Блок 1. Общие понятия 3D-печати (FDM)

Для кого: любой участник проекта с нуля. После этого блока ты понимаешь, как принтер превращает файл в деталь и о чём говорят остальные.

---

## 1.1 Принцип FDM

FDM (Fused Deposition Modeling, оно же FFF) — послойное наплавление: пластиковая нить (**филамент**, стандартный диаметр 1,75 мм) подаётся в нагретое **сопло** (~190–280 °C в зависимости от материала), плавится и выдавливается тонкой колбаской на **стол** (подогреваемую платформу). Голова движется по траектории, рисуя слой; затем принтер смещается на высоту слоя (обычно 0,1–0,3 мм) и рисует следующий поверх. Деталь растёт снизу вверх часами.

Из этого принципа следуют почти все свойства технологии:
- **Анизотропия**: деталь прочна вдоль слоёв и слаба на разрыв поперёк (межслойная адгезия всегда хуже монолита). Ориентация детали при печати — инженерное решение.
- **Нависания**: пластик нельзя класть в воздух. Нависания круче ~45–60° требуют **суппортов** (поддержек) — печатаемых подпорок, которые потом отламывают.
- **Первый слой решает всё**: он соединяет деталь со столом. Плохой первый слой = деталь отклеится посреди печати (см. блок 7 — на этом стоит наш чекпоинт).

## 1.2 Анатомия принтера

- **Хотэнд** — нагревательный узел: нагреватель + термистор (датчик температуры) + сопло. Сопло — сменная деталь с отверстием (стандарт 0,4 мм; для скорости/прочности ставят 0,6–0,8).
- **Экструдер** — механизм подачи филамента (мотор + прижимные шестерни). **Директ** — экструдер прямо на голове (наш случай: K1-семейство), лучше контроль, обязательный для гибких материалов. **Боуден** — экструдер на раме, нить идёт к голове по трубке: легче голова, хуже контроль.
- **Стол (bed)** — алюминиевая плита с нагревателем (40–110 °C) и съёмной **печатной поверхностью** (см. блок 3 и 5 — адгезия это наша ключевая тема).
- **Кинематика** — схема движения осей:
  - **Bedslinger** (Ender 3, Bambu A1): стол ездит вперёд-назад (ось Y), голова — влево-вправо и вверх. Просто, но стол с деталью мотается — ограничивает скорость.
  - **CoreXY** (наши K1C/K1/K2/K1 Max, Bambu X1/P1): голова движется в плоскости XY двумя перекрёстными ремнями, **стол только опускается вниз по Z** по мере роста детали. Быстро, точно, обычно в закрытом корпусе. Важное следствие для нас: в начале печати стол вверху (близко к камере), в конце — внизу.
- **Закрытая камера** — корпус, держащий тёплый воздух (35–60 °C). Критично для ABS/ASA (иначе коробление), безразлично для PLA (ему наоборот нужно охлаждение).
- **Вентиляторы**: обдув детали (охлаждает свежий пластик — качество нависаний), обдув хотэнда (защита от перегрева зоны выше сопла), у закрытых — вентиляция камеры.

## 1.3 Путь от модели к детали

```
3D-модель (STL/STEP/3MF) → СЛАЙСЕР (+профиль настроек) → G-code → принтер исполняет
```

1. **Модель**: STL — просто сетка треугольников (стандарт де-факто); STEP — точная CAD-геометрия; 3MF — современный контейнер (модель + настройки + метаданные).
2. **Слайсер** режет модель на слои и прокладывает траектории, генерируя **G-code** — текстовый файл команд («нагрей сопло до 220», «поедь в X10 Y20, выдавливая 0,05 мм³/мм»). Подробно — блок 2.
3. Принтер тупо исполняет G-code строка за строкой. Всё «умное» происходит до печати (слайсинг) или рядом с ней (наша система мониторинга).

## 1.4 Словарь, без которого не понять ни один разговор

| Термин | Что это |
|---|---|
| **Слой (layer height)** | Толщина одного слоя, 0,1–0,3 мм. Тоньше = качественнее и дольше |
| **Периметры (walls)** | Контурные линии слоя — стенки детали (обычно 2–4) |
| **Заполнение (infill)** | Внутренняя решётка детали, 10–40% обычно; 100% = монолит |
| **Скирт (skirt)** | Обводка вокруг детали в начале — прочищает сопло, показывает адгезию |
| **Брим (brim)** | Юбка, приклеенная к детали по периметру первого слоя — увеличивает пятно контакта, борьба с отклеиванием углов |
| **Рафт (raft)** | Печатная подложка под всей деталью — крайняя мера адгезии |
| **Суппорты** | Поддержки нависаний, отламываются после печати |
| **Ретракт (retraction)** | Втягивание нити при холостых перемещениях — борьба с соплями |
| **Стрингинг (stringing)** | Паутина/сопли между частями детали (перегрев, влажный пластик, плохой ретракт) |
| **Варпинг (warping)** | Коробление: углы детали отрываются от стола из-за усадки при остывании. Главный враг ABS |
| **Слоновья нога (elephant foot)** | Расплющенный первый слой — деталь шире у основания |
| **Z-offset** | Микрорасстояние сопло↔стол на первом слое. Сотые доли мм решают: слишком близко — задиры, слишком далеко — не прилипнет |
| **Bed mesh / автокалибровка** | Промер кривизны стола датчиком и программная компенсация при печати первого слоя |
| **Спагетти** | Авария: деталь оторвалась/сместилась, принтер часами выдавливает пластик в воздух — клубок нитей. Классика ночного брака, цель №1 нашей CV |
| **Layer shift (сдвиг слоёв)** | Авария: пропуск шагов мотора/проскок ремня — верх детали смещён относительно низа |
| **Blob of doom** | Авария: пластик налипает на сопло/хотэнд комом, может утащить провода и убить голову |
| **Экструзия/недоэкструзия** | Подача пластика; недоэкструзия (частичное забитие сопла, пережатая трубка, влажный пластик) = дырявые слабые слои |
| **Cold pull** | Чистка сопла: нагрел-остудил-выдернул нить с грязью |
| **Наработка (моточасы)** | Часы печати принтера — база регламентов ТО (блок 6) |

## 1.5 Что считается «хорошей печатью»

- Первый слой: равномерная «примятая» полоса без волн, зазоров и задиров.
- Стенки: гладкие, слои читаются равномерно, без «пропечатки» инфилла.
- Геометрия: размеры в допуске (типично ±0,1–0,3 мм на настроенной машине), без слоновьей ноги.
- Верхние поверхности: закрытые, без дыр и «подушки».
- Для серийного производства (наш случай) главное — **повторяемость**: сотая деталь = первой. Достигается зафиксированными профилями, стабильным материалом (сушка!) и регламентным ТО, а не «магией оператора».

## 1.6 Ферма — это не «много принтеров», а конвейер

Ферма 3D-печати — производство, где узкие места смещаются с «напечатать» на всё вокруг: поставить задание, снять деталь, перезапустить, отследить брак, учесть продукцию, обслужить парк. Один оператор физически обслуживает конечное число машин (замены катушек, съём, реакции на сбои) — поэтому автоматизация цикла (наша система) меняет экономику фермы сильнее, чем покупка более быстрых принтеров. Подробно про фермскую специфику — блок 7.
