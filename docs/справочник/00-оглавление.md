# Справочник по 3D-печати для команды ЛЁН

Назначение: вкатить любого участника (и любой AI-инструмент) в предметку с нуля до уровня, на котором понятны все решения ТЗ. Читается последовательно; ПРО-блок — после базы.

| Блок | Файл | Для кого обязателен |
|---|---|---|
| 1. Общие понятия | `01-общие-понятия.md` | все |
| 2. Софт и архитектура работы | `02-софт-и-архитектура.md` | Ф, Д; И — раздел про макросы |
| 3. Железо и составляющие | `03-железо-и-составляющие.md` | И, Ф |
| 4. Сеть и подключения | `04-сеть-и-подключения.md` | Ф; И — разделы MQTT и подключений |
| 5. Материалы и свойства | `05-материалы-и-свойства.md` | все |
| 6. Регламенты и обслуживание | `06-регламенты-и-обслуживание.md` | все, кто касается принтеров |
| 7. ПРО-уровень: фермы и безлюдная печать | `07-про-уровень-фермы.md` | команда, после блоков 1–6 |

Правила поддержки справочника:
- справочник — живой документ: узнали новое на стенде/пилоте (особенно из Э1–Э6) — дополняем соответствующий блок, изменение фиксируется в `CHANGELOG.md`;
- цифры в таблицах (температуры, времена) — ориентиры; рабочая истина — версионируемые профили и паспорта из `hardware/experiments/`;
- противоречие справочника и ТЗ трактуется в пользу ТЗ (справочник — обучение, ТЗ — требования).
