# ЛЁН — автоматизация ферм 3D-печати

Программно-аппаратный комплекс: от загрузки модели до готовой детали в таре — библиотека изделий, серверный слайсинг, очередь, CV-контроль качества (включая чекпоинт первого слоя), автоснятие деталей, сортировка годен/брак, трассируемость. Цель MVP — **ночная смена фермы без человека** на стойке автономии (Creality K1-парк) + софт-контур на всей площадке.

## С чего начать
1. **`AGENTS.md`** — правила проекта: конвенции, статусы, принципы, глоссарий. Обязательно к прочтению людям и AI-инструментам.
2. **`docs/ТЗ-1-система-ЛЁН-MVP.md`** — что строим.
3. **`docs/план-этапов-проекта.md`** — когда и кто.
4. **`docs/ТЗ-2-прототип-стенд-и-стойка-автономии.md`** — железо и стенд.
5. **`CHANGELOG.md`** — текущее состояние работ и статусы.

## React-прототип панели

Новый golden screen и постоянная оболочка панели находятся в `src/`. Локальный запуск:

```bash
pnpm install
pnpm dev
```

Проверки: `pnpm typecheck`, `pnpm lint`, `pnpm test`, `pnpm build`. Дизайн-контракт описан в `docs/UI-дизайн-система-панели.md`. Старые standalone-макеты временно доступны в `public/legacy/` и будут удалены после достижения паритета всех 13 маршрутов.

## Структура
```
AGENTS.md          правила проекта (единый свод; CLAUDE.md/GEMINI.md указывают сюда)
CHANGELOG.md       журнал изменений со статусами тестирования
docs/              документация (ТЗ, план, домен-модель, UX-видение, анализ рынка)
docs/справочник/   справочник по 3D-печати — вход в предметку для людей и AI (блоки 1–7)
src/               React/Vite-панель: app shell, страницы, UI/domain-компоненты, typed mocks
public/legacy/     временная копия прежних HTML-прототипов на период миграции
hardware/          рабочая зона инженера: CAD, чертежи, прошивки ESP32, макросы Klipper,
                   докшиты, протоколы экспериментов; hardware/integration/ — контракт железо↔софт
```

## Команда и зоны
- **Ф** — бэк + ML/CV (домен-модель, edge/центр, CV/VLM, слайсер-сервис)
- **Д** — дизайн + фронт (UX-видение, панель React SPA/PWA)
- **И** — инженер (стенд, сниматель, сортировка, клей-модуль, стойка)
